## 什么是Cloud Native Build？

**Cloud Native Build**（简称CNB）是一个国内的云原生开发平台，它提供了一站式的代码托管和部署服务。

### 核心特点：
- 🎯 **一站式服务**：同时提供代码托管和静态网站托管功能
- 🌐 **国内服务**：完全国内托管，访问速度快，不受国际网络影响
- 🛠️ **功能完整**：提供类似GitHub的代码托管功能
- 💸 **成本低**：基本功能完全免费

---

## cnb.cool的主要功能

### 1. 代码托管

cnb.cool提供了完整的Git代码托管功能，包括：
- 📁 **仓库管理**：创建、管理和分享代码仓库
- 🔄 **版本控制**：完整的Git版本控制功能
- 👥 **协作开发**：支持多人协作开发
- 📋 **Issue跟踪**：使用Issue功能跟踪项目问题和任务
- 🔀 **Pull Request**：使用Pull Request进行代码审查和合并

### 2. 静态网站托管

cnb.cool内置了静态网站托管功能：
- 🚀 **一键部署**：简单几步即可部署网站
- 🌍 **CDN加速**：提供快速的内容分发网络
- 🔗 **自定义域名**：支持绑定自定义域名
- 📊 **访问统计**：查看网站访问数据

### 3. 其他功能

- 🔐 **访问令牌**：使用令牌进行安全认证
- 📦 **制品仓库**：支持Docker、Maven、npm等多种制品格式
- 🤖 **AI助手**：集成AI助手辅助开发
- 💻 **在线IDE**：提供在线开发环境

---

## 为什么选择cnb.cool？

经过评估，我们选择了**cnb.cool**作为我们的平台，原因如下：

### 1. 一站式服务
- 不需要在多个平台之间切换
- 代码托管和网站部署在同一个地方
- 管理更简单，操作更方便

### 2. 国内访问稳定
- 完全国内服务器，访问速度快
- 不受国际网络波动影响
- 用户体验更好

### 3. 功能完整
- 提供完整的Git代码托管功能
- 内置静态网站托管
- 支持常见的开发工作流

### 4. 免费使用
- 基本功能完全免费
- 适合个人和小型项目
- 降低了使用门槛
