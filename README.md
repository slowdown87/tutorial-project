# 教程项目

这是一个关于如何使用Solo Trae Web Code制作网页并发布的教程项目。

## 项目结构

```
tutorial-project/
├── project/                # 实际的网页项目代码
│   ├── css/                # CSS样式文件
│   └── js/                 # JavaScript脚本文件
├── docs/                   # 教程文档
├── index.html              # 网页主文件（根目录，用于GitHub Pages）
└── README.md               # 项目说明
```

## 如何使用

1. 打开 `index.html` 文件查看网页
2. 查看 `docs/` 目录中的教程文档

## 技术栈

- HTML5
- CSS3
- JavaScript

## 项目配置

- **package.json**：项目配置文件
- **.gitignore**：Git忽略文件配置
- **README.md**：项目说明文件

本项目使用 GitHub 进行部署：
- **代码托管**：GitHub

## 如何运行

### 在线方式
1. 在 Solo Trae Web Code 中打开项目
2. 查看 `index.html` 文件

### 本地方式
1. 克隆项目到本地
2. 直接在浏览器中打开 `index.html` 文件
